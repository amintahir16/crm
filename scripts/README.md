# Scripts Directory

This directory contains utility scripts for the housing society management system.

## Manual Plot Addition

Plots are now added manually through the web UI:

1. Navigate to `/dashboard/plots/map`
2. Click "Add Plot" button
3. Click on the map image where the plot is located
4. Fill in plot details in the form
5. Save - the plot will be stored with pixel coordinates

The plot map image (`plot-map.png`) should be placed in the `frontend/public/` directory.
